# APK Installer

**APK Installer** is a PowerShell-based tool designed to simplify the installation, upgrade, and uninstallation of **Sysmon** and **UF** on Windows systems. It provides an automated and user-friendly approach for managing these security and monitoring tools.

---

## Features

- **Install**: Automatically installs Sysmon and UF with predefined configurations.  
- **Upgrade**: Checks for existing installations and upgrades to the latest version.  
- **Uninstall**: Removes Sysmon and UF cleanly from the system.  
- **PowerShell-based**: Fully scripted, no additional dependencies required.  

---

## Requirements

- Windows 10 or later  
- PowerShell 5.1 or higher  
- Administrator privileges (required for installing or uninstalling Sysmon and UF)
